Sleek
=====

Sleek is a lightweight static blog generator using Java, Spring Boot, Undertow and Gradle. 

[![Build Status](https://travis-ci.org/tjanek/sleek.svg?branch=master)](https://travis-ci.org/tjanek/sleek)
